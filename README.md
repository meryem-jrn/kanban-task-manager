# Kanban Task Manager

## Sujet du projet
Application web de gestion de tâches inspirée d’un tableau Kanban.
Elle permet de créer, modifier, supprimer et déplacer des tâches,
et de les associer à des utilisateurs.

## Technologies utilisées
- React
- Redux Toolkit
- React Router
- JSON Server
- JavaScript ES6+
